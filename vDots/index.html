Installation

1

Clone the Repository

git clone https://github.com/viv223345/vDots.git ~/.config/hypr
2

Install Dependencies

# Arch Linux/Manjaro
sudo pacman -S hyprland kitty waybar wofi dunst swaylock-effects \ swaybg wl-clipboard polkit-gnome network-manager-applet \ xdg-desktop-portal-hyprland
3

Apply Configuration

# Backup your existing config first (if needed)
mv ~/.config/hypr ~/.config/hypr.bak

# Link or copy the repository content
ln -s ~/path/to/vDots ~/.config/hypr

# Make scripts executable
chmod +x ~/.config/hypr/scripts/*
4

Start Hyprland

Log out of your current session and select Hyprland from your display manager, or start it directly:

exec Hyprland
